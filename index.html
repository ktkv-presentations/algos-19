CLI

Интерфейс командной строки

способ взаимодействия с программами через текстовые команды, вводимые пользователем в терминале или консоли

Текстовый пользовательский интерфейс

использует псевдографику для организации интерфейса в текстовом режиме терминала

Примеры TUI программ

GUI

Graphical User Interface

Графический интерфейс пользователя

способ взаимодействия пользователя с программой с помощью визуальных элементов

Основные элементы GUI

  • Окно (Window) – главный контейнер, в котором располагаются другие элементы.
  • Кнопка (Button) – интерактивный элемент, запускающий действие при нажатии.
  • Метка (Label) – текстовая информация, которая отображается на экране.
  • Поле ввода (Entry, Input Field) – позволяет пользователю вводить текст.

Основные элементы GUI

  • Флажок (Checkbox) – элемент для выбора нескольких опций.
  • Переключатель (Radio Button) – позволяет выбрать только один вариант из нескольких.
  • Выпадающий список (Dropdown, ComboBox) – список возможных вариантов для выбора.
  • Поле для вывода текста (Text Area) – многострочное поле для ввода и редактирования текста.
  • Меню (Menu Bar) – верхняя панель с пунктами меню (Файл, Правка и т. д.).

Tkinter

стандартный пакет для создания графического интерфейса (GUI) в Python

является обёрткой над Tcl/Tk – кроссплатформенным инструментом для работы с GUI


import tkinter as tk

Основные преимущества Tkinter

  • Встроен в Python
  • Прост в использовании
  • Кроссплатформенность
  • Гибкость

Работа с Tkinter

Создание окна в Tkinter


import tkinter as tk

# Создаём главное окно

# Устанавливаем заголовок окна

Приложения с GUI реагируют на действия пользователя (клики, ввод текста, перемещение мыши).

Пример обработки клика по кнопке


def on_click():
    print("Кнопка была нажата!")

btn = tk.Button(root, text="Кликни меня", command=on_click)

Обработка событий (Event Handling)

Приложения с GUI реагируют на действия пользователя (клики, ввод текста, перемещение мыши).

Пример обработки нажатия клавиши


def key_pressed(event):
    print(f"Нажата клавиша: {event.char}")

root.bind("", key_pressed)
  • <Button-1> – клик левой кнопкой мыши
  • <Button-3> – клик правой кнопкой мыши
  • <Key> – любое нажатие клавиши
  • <Return> – нажатие Enter
  • <Motion> – перемещение мыши 

import tkinter as tk

def on_button_click():
    text = entry.get()  # Получаем текст из поля ввода
    label.config(text=f"Вы ввели: {text}")  # Обновляем текст метки

# Создаём главное окно
root = tk.Tk()
root.title("Простое приложение")  # Заголовок окна
root.geometry("300x150")  # Размер окна

# Создаём виджеты
label = tk.Label(root, text="Введите текст:")
label.pack(pady=5)  # Размещаем с отступом

entry = tk.Entry(root)
entry.pack(pady=5)

button = tk.Button(root, text="Показать", command=on_button_click)
button.pack(pady=5)

# Запускаем главный цикл
root.mainloop()
